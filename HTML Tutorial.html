Introduction


	HTML was originally designed to be very simple and handle text-based documents.

	As the web became popular and browsers developed, new features were added to subsequent versions.
	
	Structure is important in HTML for a number of reasons:
	–Allows for consistent rendering in browsers
	–Means users can take advantage of accessibility features for display and navigation of pages
	–Makes editing and maintaining documents easier
	–Gives page authors full access to style sheet and scripting features


Basic HTML Document Structure 

	An HTML document is composed of three parts:
	1. A line containing HTML version information
	2. A declarative header section (delimited by the <head> element)
	3. A body, which contains the document's actual content…implemented by the <body> element…

	Sections 2 and 3 should be delimited by the <html> element
			-http://www.w3.org/TR/html401/struct/global.html


Head

	The <head> element is a container for all the head elements.

	The <head> element can include a title for the document, scripts, styles, meta information, and more.

	The following elements can go inside the <head> element:

	<title> (this element is required in an HTML document)
	<style>
	<base>
	<link>
	<meta>
	<script>
	<noscript>
	
	Here you can find the W3School page on the head tag Click me


Body

	The <body> tag defines the document's body.

	The <body> element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
	
	Its final appearance may be influenced by information from the <head> (scripts, style sheets etc)
	
	
	
	Here you can find the W3School page on the body tag Click me


Tags

	Tags are wrapped around < and > and they can be used inside the head as well as the body.

	The <header> tag,for example, represents a container for introductory content or a set of navigational links.	
	
	

	You can have several elements with the same tag in one document, multiple h1 tags for example.
	
	Some tags are not paired (never enclose any content), such as </br> which is used to go to the next line.
	
	
	Here you can find the W3School page on tags Click me


Attributes

	Attributes provide additional information about HTML elements.
	
	They are declared in the opening tag.
		<div id="content-block-1">content</div>
	
	Self-closing tags can contain attributes too.
		<img src="logo.gif" alt="My Logo" />

	Next I will give example of some attributes.
	
	The href Attribute:
	HTML links are defined with the <a> tag. The link address is specified in the href attribute:
	<a href="https://www.google.com">This is a link</a>

	The src Attribute
	HTML images are defined with the <img> tag.
	The filename of the image source is specified in the src attribute:
	<img src="Puppy image.jpg">
	
	
	The title Attribute
	The title attribute provides a descriptive tag for the image when hovered over
	<img src="Puppy image.jpg" title="Picture of a puppy">
	
	
	The width and height Attributes
	Images in HTML have a set of size attributes, which specifies the width and height of the image:
	<img src="Puppy image.jpg" width="300" height="200">
	
	
	The alt Attribute
	The alt attribute specifies an alternative text to be used, when an image cannot be displayed.
	The value of the attribute can be read by screen readers. This way, someone "listening" to the webpage, e.g. a vision impaired person, can "hear" the element.
	<img src="wrong image name.jpg" alt="Non-existent image">

Tables

	Tables have been introduced into HTML to describe tabular data
	An HTML table is defined with the <table> tag.
	Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. 
	By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.
	
	
	If you do not specify a border for the table, it will be displayed without borders.
